# Hero_legend
🗡️ **Hero Legend** - Game aksi 2D platformer epik berbasis browser.  Lawan musuh &amp; bos legendaris di 5 level bertema unik, kumpulkan senjata dan skill baru, serta taklukkan Endless Mode yang semakin sulit.  Fitur: Combo seru, animasi smooth, particle effects, double jump, dash, dan boss battle epik. Full Vanilla JavaScript + HTML5 Canvas.
